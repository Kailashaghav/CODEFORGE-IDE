# ⚡ CodeForge IDE

> A lightweight, dark-themed Java code editor built with JavaFX — featuring syntax highlighting, a terminal-style interface, and a smooth developer experience.

![Java](https://img.shields.io/badge/Java-21-orange?style=flat-square&logo=openjdk)
![JavaFX](https://img.shields.io/badge/JavaFX-21-blue?style=flat-square)
![Maven](https://img.shields.io/badge/Maven-3.9+-red?style=flat-square&logo=apachemaven)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

## ✨ Features
- 🎨 Syntax Highlighting — keywords, strings, comments, numbers
- 🌑 Dark Theme — easy on the eyes
- 🖥️ Built-in Terminal — run commands inside the editor
- 📦 Single Executable JAR — package and share with one command

## 🛠️ Tech Stack
| Technology | Version |
|------------|---------|
| Java | 21 |
| JavaFX | 21 |
| RichTextFX | 0.11.3 |
| Maven | 3.9+ |

## 🚀 Run the Project
```bash
git clone https://github.com/YOUR_USERNAME/codeforge-ide.git
cd codeforge-ide
mvn clean javafx:run
```

## 📦 Build a JAR
```bash
mvn package -Pfat-jar
java -jar target/codeforge-ide-1.0.0-shaded.jar
```
